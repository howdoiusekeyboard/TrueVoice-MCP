TrueVoice MCP

Tools to eliminate AI slop from text

What is this

MCP server that eliminates AI slop. Based on research from NLP writers and philosophers. arXiv:2509.19163v1

Setup

Add to Cursor Add to VS Code
  1. Open Claude Desktop
  2. Click + in the chat bar, then Connectors
  3. Click Add custom server
  4. Paste this URL:
    https://truevoice-mcp.kushagragolash.tech/api/mcp

Using Windsurf, Zed, or another client? See the tab for the server URL.

Server URL

https://truevoice-mcp.kushagragolash.tech/api/mcp

MCP Configuration

{
  "mcpServers": {
    "truevoice": {
      "url": "https://truevoice-mcp.kushagragolash.tech/api/mcp"
    }
  }
}
